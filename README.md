**Server Status**<br/>

##### Description 
Classic game of [Pong](https://en.wikipedia.org/wiki/Pong "Pong Wikipedia page") but played using hand detection 
on your webcam (move hand up or down to move the pallet)

##### Usage

1. Install
2. Play
